시스템프로그래밍 프로젝트 #2 <20141583 정현빈>

본 프로젝트는 1주차에서 작성한 SIC Simulator에 더하여 type, assemble, symbol의 3가지 기능을 추가하는 것이 목적입니다.

make 입력 시 코드가 컴파일되어 20141583.out 실행파일이 생성되며
./20141583.out 입력 시 실행파일이 실행되어 sicsim의 대기화면으로 진입합니다.



sicsim에서 입력 가능한 명령어의 리스트는 help 명령어 입력 시 확인할 수 있습니다.



sicsim 명령어 입력 창에서 quit 입력 시 sicsim이 종료되며 
make clean 입력 시 20141583.out 실행파일이 삭제됩니다.
