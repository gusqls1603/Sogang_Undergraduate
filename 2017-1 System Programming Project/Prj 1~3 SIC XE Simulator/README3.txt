시스템프로그래밍 프로젝트 #3 20141583 정현빈

이 프로젝트는 SIC/XE 가상머신으로 어셈블러, 링커, 로더 및 실행기능을 구현하는 것이 목적입니다.

make 입력 시 gcc -Wall 옵션으로 20141583.c 파일이 컴파일되며, ./20141583 입력을 통해 SIC/XE 머신의 실행이 가능합니다.

3차 프로젝트에서 구현된 명령어는 progaddr, loader, run, bp 등 4개이며

 각 명령어의 입력 포맷은 help 명령어 입력 시 확인이 가능합니다.


프로그램 종료는 quit 입력으로 가능하며, SIC/XE 머신 종료 뒤 make clean 명령어 입력 시 실행파일이 삭제됩니다.
